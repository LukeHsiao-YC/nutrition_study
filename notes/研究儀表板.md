# 📊 研究儀表板

> 這頁的表格需要 **Dataview** 外掛才會顯示(安裝步驟見 [[../docs/OBSIDIAN|OBSIDIAN.md]] 或 docs/OBSIDIAN.md)。
> 表格會即時反映 `src/content/articles/` 裡所有研究卡的 frontmatter,不用手動維護。

---

## 🔴 待確認清單(AI 草稿還沒經你確認)

```dataview
TABLE WITHOUT ID file.link AS 論文, journal AS 期刊, study_design AS 設計, grade.certainty AS GRADE
FROM "src/content/articles"
WHERE depth = "reading" AND (pico.confirmed = "" OR grade.confirmed = "")
SORT pubDate DESC
```

## 📚 深讀庫總覽(依證據等級排序)

```dataview
TABLE WITHOUT ID file.link AS 論文, rob.tool AS RoB工具, grade.certainty AS GRADE, sample_size AS 樣本
FROM "src/content/articles"
WHERE depth = "reading" OR depth = "done"
SORT grade.certainty ASC
```

## 🏷️ 依主題(標籤)分組

```dataview
TABLE rows.file.link AS 論文
FROM "src/content/articles"
WHERE depth = "reading" OR depth = "done"
FLATTEN tags AS 主題
GROUP BY 主題
```

## 📈 各期刊深讀篇數

```dataview
TABLE length(rows) AS 篇數
FROM "src/content/articles"
WHERE depth = "reading" OR depth = "done"
GROUP BY journal AS 期刊
```

## ✅ 已完成(depth = done)

```dataview
LIST
FROM "src/content/articles"
WHERE depth = "done"
```

---

### 使用小技巧
- 想改條件?把上面的 `WHERE`/`SORT` 換掉即可(例:`WHERE grade.certainty = "High"` 只看高證據等級)。
- **關聯圖**:左側開「關聯圖檢視」→ 右上設定裡打開「標籤(tags)」,共用標籤的論文就會自動群聚,一眼看出主題群與缺口。
